# Snake Game (贪吃蛇游戏)

一个用纯 JavaScript 开发的经典贪吃蛇游戏。

## 游戏特性

- 🎮 流畅的蛇移动和控制
- 🍎 随机生成的食物
- 💥 碰撞检测（墙壁和自身）
- 🔄 游戏结束自动重新开始
- ⌨️ 键盘控制（方向键）

## 如何运行

1. 打开 `index.html` 文件在浏览器中
2. 使用方向键控制蛇的移动
3. 吃掉红色食物让蛇变长
4. 避免撞墙或撞到自己

## 游戏规则

- 使用方向键控制蛇的移动方向
- 吃掉红色方块食物可以让蛇变长
- 撞到墙壁或蛇身会导致游戏结束
- 游戏结束后会自动重新开始

## 文件结构

- `index.html` - 游戏的主页面
- `style.css` - 游戏样式
- `game.js` - 游戏主循环和逻辑
- `snake.js` - 蛇的移动和绘制
- `input.js` - 键盘输入处理
- `food.js` - 食物生成和碰撞检测

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
